# Car Safety Guardian

A simple browser-based safety app that simulates monitoring a locked car for a child left behind.

## Protected copy notice

This app and its content are provided as a protected demonstration. Please do not copy, redistribute, or republish it without authorization.

## Run locally

Open [index.html](index.html) in a browser, or serve the folder with a simple static server:

```bash
python -m http.server 8000
```

Then open http://localhost:8000/ in your browser.
